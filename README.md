# 📝 Todo Tasks App

Современное приложение для управления задачами с приоритетами, календарём и таймером Pomodoro.

## 🎯 О проекте

Todo Tasks - это полнофункциональное приложение для управления задачами, которое помогает организовать рабочий процесс и повысить продуктивность. Приложение сочетает в себе классический todo-лист, календарь для планирования и технику Pomodoro для эффективной работы.

## 🚀 Основные возможности

- ✅ **Создание и управление задачами** - добавление, редактирование и удаление задач
- 🎨 **Система приоритетов** - low, medium, high для важных задач
- 📅 **Календарь** - планирование задач по дням недели
- ⏱️ **Pomodoro таймер** - техника для концентрации на работе
- 🔐 **Аутентификация** - личный кабинет для каждого пользователя
- 💾 **Синхронизация данных** - сохранение в базе данных PocketBase
- ✨ **Плавные анимации** - приятный UX с Framer Motion

## 🛠️ Технологии

### Frontend
- **React 19** - современная библиотека для UI
- **TypeScript** - типизация для надёжного кода
- **Vite** - быстрая сборка и HMR
- **React Router** - маршрутизация и навигация
- **Tailwind CSS** - utility-first стили
- **Framer Motion** - плавные анимации
- **React Icons** - иконки для интерфейса

### Backend
- **PocketBase** - легковесная база данных и аутентификация
- REST API для работы с задачами

### Инструменты разработки
- **ESLint** - линтинг кода
- **TypeScript ESLint** - правила для TypeScript
- **Vite Plugin React** - Fast Refresh

## 🎨 Особенности реализации

### Управление состоянием
- **Context API** для глобального состояния задач
- Кастомные хуки для логики (useTasks, useUI, useUser)
- LocalStorage для кеширования данных

### Анимации
- Плавные переходы с Framer Motion
- Анимация появления/исчезновения модальных окон
- Интерактивные элементы с hover эффектами

### Типизация
- Строгая типизация с TypeScript
- Интерфейсы для всех данных
- Type-safe API запросы
